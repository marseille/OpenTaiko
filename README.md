# OpenTaiko
Personal unofficial build of OpenTaiko. It supports some features not found in the officially supported version, and intentionally held back. Note there is no support provided for this code, everything here is provided on AS-IS condition. In addition, look else-where for songs and/or skins related to this game. They aren't included here.

For the official and supported distribution of OpenTaiko, with loads more quality assurance, look here: https://github.com/0auBSQ/OpenTaiko

## Features available
* favorites for player-two
  * graphics + sound
  * sorting
* Condensed main-menu for only arcade play
* Graphic for new highscore on result screen
* Increased fast-skip on song-select
